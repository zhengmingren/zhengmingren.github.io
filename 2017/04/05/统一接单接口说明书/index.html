13. 获取门店的菜单

获取门店的菜单(含分类)

请求地址

http://123.57.147.102:10000/takeout-app-serv/comm

请求方法

getDishList

输入

名称 类型 是否必须 说明
en string 是 en号
storeCode string 是 旺POS门店编码

输出

名称 类型 是否必须 说明
status int 是 状态码，0：成功，-1：失败
info string 是 状态说明
data Map 是 菜单Map,key为分类名,value为该分类下的所有菜品(Dish)的List列表

Dish模型

名称 类型 是否必须 说明
name string 是 菜品名称
dish_id arry 是 菜品ID
price string 是 菜品价格
category_name double 是 菜品分类
pictureUrl double 是 菜品图片
status double 是 菜品状态,0为上架状态,1为下架状态
type double 是 菜品类型,0为普通菜品,1为套餐菜品,2为商品

附录

订单状态说明

状态码 状态说明
0 平台生成推送订单
1 POS已接受订单
2 POS已拒绝接受
3 第三方平台取消

申请退单状态说明

状态码 状态说明
0 未申请退单
1 用户申请退单
2 退单成功
3 不同意退单
4 退单仲裁中
5 退单失败
6 用户取消退款申请
7 取消退款申诉

配送状态说明

状态码 状态说明
0 初始化状态
1 待分配（物流系统已生成运单，待分配配送商）
2 待分配（配送系统已接单，待分配配送员）
3 待取餐（已分配给配送员，配送员未取餐）
4 配送中（配送员已取餐，正在配送）
5 配送成功（配送员配送完成）
6 配送取消（商家可以重新发起配送）
7 配送异常
